PORK PAY — BACKGROUND IMAGES
============================

Drop two image files in THIS folder, named exactly:

    john-pork.jpg          <- shows on the LEFT half of the background
    daquavious-pork.jpg    <- shows on the RIGHT half of the background

Then refresh the site (Ctrl+Shift+R) and the backgrounds appear behind the swap box.

Notes:
- .jpg, .png, or .webp all work, but the filename must match (rename to .jpg
  or tell Claude if you use a different extension).
- Landscape/portrait both fine — they're set to "cover" so they fill the half.
- Until you add them, the background just shows dark pork color (not broken).

Easiest way to get the images:
- Google "John Pork" and "Daquavious Pork", right-click an image you like,
  "Save image as...", into this folder with the names above.
- OR paste the image URLs to Claude and it'll download them for you.
